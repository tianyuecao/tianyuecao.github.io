KITTI to VOC

Codes are available at https://github.com/tianyuecao/KITTI2VOC.

  1. Make new folders ‘Annotations’(save xml files), ‘ImageSets’(save a ‘Main’ folder to save train.txt, test.txt and val.txt) and ‘JPEGImages’(save all labed images).
  2. Transfer categories, use the file ‘modify_annotations_txt.py’, modify the original KITTI labels to the labels we need. (note that the original files will be modified, so be careful to backup the files.
  3. Transfer txt label files to xml format and save to ‘Annotations’ folder. Use the file ‘txt_to_xml.py’.
  4. Generate train/val/test lists. Use create_train_text_txt.py, save to the folder ‘ImageSets/Main’.
  5. According to Annotation, use ‘create_kitti_label.py’, generate the final labels fils and training and testing lists.
