What is algorithm

An algorithm is a sequence of unambiguous instructions for solving a problem, i.e., for obtaining a required output for any legitimate input in a finite amount of time.

Euclid’s Algorithm

Problem

求gcd(m,n)，两个非负整数的最大公因数，并且不是0且不是m和n。
e.g. gcd(60,24) = 12, gcd(60,0) = 60, gcd(0,0) = ?

Analysis

Euclid’s algorithm基于gcd(m,n) = gcd(n, m mod n)，重复直至第二个参数为0。
e.g. gcd(60,24) = gcd(24,12) = gcd(12,0) = 12

伪代码

另两种方法计算gcd(m,n)

Consecutive integer checking algorithm

  1. t赋值为min{m,n}
  2. 用m除以t，若余数为0，转到3；否则转到4
  3. 用n除以t，若余数为0，返回t并停止；否则转到4
  4. t=t-1，转到2.

Middle-school procedure

  1. 找到m的质因数
  2. 找到n的质因数
  3. 找到所有共同的质因数
  4. 计算所有共同质因数的乘积即位gcd(m,n)
